# Социальная сеть

## Использованные технологии:

- ReactJS
- Redux
- React-redux
- Redux-thunk
- React-router-dom
- Redux-form
- Axios

- Редактирование профиля
- Редактирование статуса
- Список всех зарегистрированных пользователей
- Пагинация
- Добавление/удаление пользователя в список друзей
